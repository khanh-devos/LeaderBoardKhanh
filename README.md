## 💻 Getting Started <a name="getting-started"></a>

- Open your Terminal
- Navigate to the desired directory to which you want to clone the repository to, then clone it down to your computer.

### Prerequisites

In order to run this project you need:
- Basic knowledge of HTML and CSS
- Basic knderstanding of Linters
- Basic understanding of Github
- Your machine should have Chrome and VSCode installed 

### Setup

Clone this repository to your desired folder:

- git clone https://github.com/khanh-devos/LeaderBoardKhanh.git

### Install

Install this project with:

- You need to install Chrome if you want to see it working locally

### Usage

To run the project, execute the following command:

- open index.html with Chrome or other browsers.

### Run tests

To run tests, run the following command:

- npx stylelint "\*_/_.{css,scss}"
- npx hint .
- npx eslint .

### Deployment

You can deploy any service you want.
